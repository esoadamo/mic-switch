# Mic Switch

## Usage

### Install and run

```bash
npm install
npm start
```

Will launch websocket server on port 8666 and web server on port 8665

### Profit

Just point your phone/notebook/w/e to your IP on port 8665 to use the switching feature.

[http://127.0.0.1:8665](http://127.0.0.1:8665)
